# Cozy Light

Lightweight personal cloud based on Node.js: files, calendars, contacts, todos
store on your machine and managed with web apps.

# Install 

    # Install Node.js, essential build tools then...
    npm install cozy-light -g

# Run

    cozy-light start

# Install application

    cozy-light add cozy-labs/calendar
    cozy-light add cozy-labs/tasky
